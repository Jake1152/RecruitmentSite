```
wanted pre-onboarding 사전과제
```

# 요구 사항 분석 & 구현 과정
## 문제 재정의

- 채용공고 사이트를 만든다
- 채용공고를 등록, 수정, 삭제, 검색할 수 있게 구현한다
  - 수정의 경우 회사 id외에는 전부 수정 가능
- DB는 RDB를 쓰며, ORM을 사용해야한다.
